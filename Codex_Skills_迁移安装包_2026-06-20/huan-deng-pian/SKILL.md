# 幻灯片处理

使用 python-pptx 创建、读取和编辑 .pptx 文件。

## 何时使用
- 创建或编辑 PowerPoint 演示文稿
- 从模板生成幻灯片
- 提取幻灯片内容

## 依赖
```
uv pip install python-pptx
```

## 工作流
1. 分析需求
2. 设计幻灯片结构
3. 使用 python-pptx 创建或编辑
4. 验证输出
