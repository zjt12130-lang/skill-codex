# ChatGPT 应用构建

构建、搭建、重构和故障排查 ChatGPT Apps SDK 应用，结合 MCP 服务器和小部件 UI。

## 何时使用
- 设计工具
- 注册 UI 资源
- 连接 MCP Apps 桥接
- 应用元数据和 CSP/域名设置
- 生成项目脚手架

## 工作流
1. 文档优先：先查 OpenAI 文档
2. 设计 MCP 服务器工具
3. 构建小部件 UI
4. 连接 MCP 桥接
5. 测试和调试

## 核心技术
- MCP (Model Context Protocol)
- React Web 组件
- ChatGPT Apps SDK
