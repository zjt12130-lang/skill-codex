# Figma 设计生成

将应用页面、视图或多部分布局翻译到 Figma 中。

## 何时使用
- 写入 Figma
- 从代码创建 Figma
- 将页面推送到 Figma
- 更新 Figma 屏幕匹配代码

## 工作流
1. 发现设计系统组件、变量和样式
2. 从 Code Connect 文件学习
3. 对相关变量进行变量设置
4. 创建所需的组件和变体
5. 组装视图
6. 更新设计覆盖和变体属性的交互
