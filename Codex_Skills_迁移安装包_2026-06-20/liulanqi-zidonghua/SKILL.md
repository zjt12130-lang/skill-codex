# 浏览器自动化

使用 Playwright 从终端自动化浏览器操作。

## 何时使用
- Web 自动化测试
- 网页数据抓取
- 表单自动填写
- 截图生成

## 依赖
```
npm install -g playwright
playwright install
```

## 工作流
1. 编写 Playwright 脚本
2. 运行自动化
3. 验证结果
