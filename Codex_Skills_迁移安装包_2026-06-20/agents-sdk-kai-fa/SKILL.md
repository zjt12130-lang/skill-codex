# OpenAI Agents SDK 开发

构建、运行、部署和评估 OpenAI Agents SDK 应用。

## 何时使用
- 用户要求创建或适配 Agents SDK 应用
- 从提示词构建智能体原型
- 准备可运行的原型
- 添加评估套件
- 通过 Agents SDK Deployment Manager 部署

## 核心概念
- Agent：具有指令和工具的智能体
- Handoff：智能体间交接
- Guardrails：安全护栏
- Tracing：追踪

## 工作流
1. 明确需求和范围
2. 设计智能体架构
3. 实现核心逻辑
4. 添加工具和集成
5. 测试和评估
6. 部署
