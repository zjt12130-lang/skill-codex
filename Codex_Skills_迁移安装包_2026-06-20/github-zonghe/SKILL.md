# GitHub 综合管理

使用可用的 MCP 工具集全面管理 GitHub 仓库。

## 何时使用
- 需要综合的 GitHub 仓库管理
- 高级/非标准请求
- 涉及多个领域的工作流
- 没有更专业技能的跨领域任务

## 能力
- 仓库：创建、查看、管理、分叉
- PR：创建、查看、更新、合并、审查
- Issue：创建、更新、搜索、添加评论
- 代码：搜索、查看文件
- 搜索：代码、用户、Issue、PR
- 发布：创建、管理
