# LaTeX 编译

使用适当工具链编译 LaTeX/TeX 项目生成 PDF。

## 何时使用
- 编译 LaTeX 文档
- 生成 PDF 输出
- 调试编译错误

## 工作流
1. 检测项目类型
2. 选择工具链（pdflatex/xelatex/lualatex）
3. 运行编译
4. 检查错误
5. 重复编译以解决引用
